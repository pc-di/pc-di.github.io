Page de démo et de test pour les notifications Web

Cette page ne fonctionne pas localement. Vous devez d'abord télécharger et copier à la racine de votre site Web :

  • les fichiers fournis sur la plateforme Dialog Insight
  • cette page de démonstration et de test

Ensuite visitez cette page à partir de votre site Web.

Associer un contact à une notification - onclick="DI.WebPush.Identify("f_EMail", "Valeur");"
Commencez par ajouter le contact auquel envoyer le test dans le projet Dialog Insight où sont configurés les notifications Web, tapez ensuite le code de la clé unique de votre projet ici et cliquez sur « Associer ». Vous pourrez ensuite créer une notification Web dans Dialog Insight et envoyer un test à ce contact.
